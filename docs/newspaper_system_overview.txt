# 👑 ROLE SYSTEM (FINAL)

## 👑 Super Admin (Platform Owner / Developer)
**Scope:** Entire system (all agencies)

### Permissions:
- Manage all agencies  
- View:
  - Agency analytics 📊  
  - Server performance ⚙️  
- Control:
  - Server / database  
  - Global settings  
- Enable / disable agencies  
- Monitor usage  

❌ Cannot interfere with daily agency operations (best practice)

---

## 🏢 Admin (Agency Owner)
**Scope:** Full control within their agency  

### Permissions:
- 📦 Add & manage stock  
- 👷 Add/remove workers  
- 👥 Manage customers  
- 💰 Set pricing  
- 🧾 Billing & payments  
- 📊 View reports  
- 💸 Manage salaries  

👉 ✔ Admin can do anything inside their agency  

---

## 👷 Worker (Distributor)
**Scope:** Very limited (only their data)

### Permissions:
- View:
  - Assigned newspapers  
  - Today’s stock taken  
- Enter:
  - Taken quantity  
  - Returned quantity  
- View:
  - Their sales  
  - Their salary  

❌ Cannot:
- See pricing  
- See other workers  
- Access admin data  

---

# 🔄 SYSTEM FLOW (END-TO-END)

## 🟢 Step 1: Super Admin

- Creates admin account  

---

## 🔵 Step 2: Admin Setup

  - Newspapers

  - Workers 👷  
  - Customers 👥

### Morning:
- Admin adds stock 📦  

### Distribution:
- Workers take newspapers  

### Evening:
- Workers enter returns  

---

## ⚙️ System Automatically Calculates:
- Sold = Taken − Returned  
- Daily income  
- Remaining stock  

---

## 📅 Step 4: Monthly Process
- Generate customer bills  
- Calculate worker salary  
- Show profit/loss
